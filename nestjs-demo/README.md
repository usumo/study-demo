# nestjs-demo

## Installation

```bash
$ cnpm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## create controller

```bash
$ nest g controller object-name

# 如创建用户
$ nest g controller user
```

## create module

```shell script
$ nest g module object-name

# 如创建用户
$ nest g module user
```
